# repopruebas
Repositorio para pruebas

# Arrancar entornos en terminal
## Arrancar backend
cd backend

node .\index.js

Comprobación de la misma en el navegador mediante http://localhost:3000/

## Arrancar frontend
cd frontend

ng serve

Comprobación de la misma en el navegador mediante http://localhost:4200/
